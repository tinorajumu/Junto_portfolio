# AIマスコットの音声ファイル置き場

ここに `<id>.wav`（または `.mp3` / `.ogg`）を置くと、ビルド時に自動で読み込まれ、
AIコンシェルジュマスコットがブラウザ読み上げの代わりにその音声で喋るようになります。
ファイルが無いセリフはこれまで通りブラウザの音声合成（Web Speech API）で読み上げます。

## 命名ルール

ファイル名（拡張子なし）が `src/data/mascotRules.ts` の各セリフの `id` と一致している必要があります。

現在のid一覧:

- `greeting` — 挨拶
- `profile` — プロフィール案内
- `timeline` — 年表案内
- `factory` — ファクトリー案内
- `dojo` — ドジョウの話
- `feed` — エサやりの話
- `depth` — 深海エリアの話
- `thanks` — お礼
- `utsushiyo` — UTSUSHIYOの話
- `event` — イベント情報案内
- `dislike` — けなされたときの返し
- `fallback1` / `fallback2` — 聞き取れなかったときの返し
- `intro` — マスコットを開いた時の最初のセリフ

例: 「こんにちは」に対する返事の音声なら `greeting.wav` という名前でこのフォルダに置いてください。

AIVtuberプロジェクト（白河零 / Shirakawa Rei ボイス、Qwen3-TTS + Seed-VC で生成）と同じ声を
使う想定です。生成が済んだらこのフォルダにコピーして再ビルド・再デプロイしてください。
